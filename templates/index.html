API Information
Endpoint:

POST /speak

Request Format:
{
  "text": "Your educational content here (required)",
  "lang": "Language code (optional, defaults to 'en')",
  "speed": "Speech speed: 'slow', 'normal', 'fast' (optional, defaults to 'normal')",
  "educational": "Enable educational mode for clearer pronunciation (optional, defaults to true)"
}
Response:

Returns an MP3 audio file optimized for YouTube Shorts or JSON error message with appropriate HTTP status codes.

Additional Endpoints:

GET /languages - Get list of all supported languages
